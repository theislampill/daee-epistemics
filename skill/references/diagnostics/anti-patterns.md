# Anti-Patterns

> role: self-audit
> use when: preparing, reviewing, or correcting a response path
> do not use when: the case is trivial and no diagnostic routing is being performed
> output: compact checks against misuse, forced fit, and rhetorical drift

## Core Anti-Patterns

| Anti-pattern | Symptom | Correction |
|--------------|---------|-----------|
| Diagnosis collapse | Replying to content before classifying the noetic issue | Run V1/M5 first and name the upstream issue |
| Tactic over-selection | Loading many modules because several seem relevant | Choose the smallest matched subset that changes the next move |
| Forced fit | Pushing an unfamiliar case into a familiar module | Mark the read provisional and state what would change it |
| Recursive overfitting | Re-running diagnosis after every move without a new differentiator | Reassess only when the prior move changes intervention order or exposes a new live barrier |
| Cumulative-case inflation | Adding many supporting routes while one unresolved blocker still dominates | Use E3 / V6 only after the dominant blocker has been cleared and stop once convergence is clear |
| Decorative terminology | Arabic or technical terms add prestige but no discrimination | Use the term only if it changes routing, scope, or doctrinal precision |
| Premature rebuttal | Countering before separating logical, probabilistic, moral, historical, and relational pressure | Clean the claim-type first |
| Grief/criterion confusion | Treating suffering-register as if it were criterion-protest | Test whether the case needs M4 before argument |
| Rhetorical overreach | Attributing motive, concealment mode, or discourse orientation without basis | Downgrade confidence and mark the claim as inference or leave it open |
| Inference laundering | Presenting model synthesis or inference as if the file stated it | Use the inference-boundary markers explicitly |

## Quick Self-Audit

- Have I diagnosed before rebutting?
- Am I using a term because it distinguishes, or because it sounds weighty?
- Am I forcing this case into a preferred module?
- Is the discourse orientation established or only guessed?
- Have I preserved restoration over rhetorical win?
- Have I marked where inference begins?
