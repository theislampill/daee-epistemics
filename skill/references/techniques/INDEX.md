# Techniques — Index

Techniques shape the architecture of a *stretch* of conversation. They are deployed
proactively to create conditions for genuine epistemic progress. A technique governs
a stretch; a tactic governs a move within it.

## Organisation

Techniques are organised by deployment stage: diagnostic and framework-clearing first,
then sign-direction and consolidation. Load the specific file needed.
