# Philosophical Usurpation - Named Case Family

Philosophical usurpation occurs when an imported framework has been installed as the upstream authority that revelation must satisfy before its content may be accepted, discussed, or deployed.

This is distinct from:

- simply holding a philosophical framework
- using philosophical arguments
- pressing a specific philosophical objection

Usurpation specifically means the framework has been granted the chair.

This family is not only a label. Once the tribunal has been named or loosened, refresh the
case-state before releasing content. `tribunal-loosened` and even `frame-cleared` are real
progress, but they do not yet equal `alignment-advanced`. If another live burden now governs,
route there explicitly. If not, release only the smallest restorative next move.

## Detection Hooks

Signal the family when one or more of these are active:

- a prior constraint on what God must be before revelation is heard
- a required allegiance to a philosophical standard as the test of legitimacy
- a verdict over the source, where revealed content must be reinterpreted to fit the framework
- an invisible background assumption treated as obvious
- asymmetric application of the standard
- a closed canon, charismatic disclosure, or non-Islamic ontology claiming veto power over later revelation, transmitted authority, or Creator-creation distinction

## Varieties

### Type A - Aristotelian / Neo-Platonic Theism as Default Standard

The God of immutability, impassibility, pure actuality, and simplicity is treated as the philosophically correct standard, and the God of revelation appears defective by comparison.

When this Type A pressure is specifically functioning as an imported perfection / immutability / non-eventfulness tribunal, route first to `references/diagnostics/perfection-criterion-usurpation.md`. That file is the canonical owner for the perfection-criterion family; this file remains the wider usurpation-family router.

### Type B - Scientific Naturalism as Epistemic Gate

Natural-scientific method is treated as the complete and correct epistemology.

### Type C - Historical-Critical Methodology as Hermeneutic Authority

Historical-critical method is treated as the neutral and controlling way to evaluate revelation-bearing texts.

### Type D - Liberal Political Philosophy as Moral Arbiter

Liberal autonomy or a related political-moral grammar is treated as the prior standard against which revealed ethics must answer.

### Cross-Tradition Structural Instantiations

These examples do not add new case-family coverage. They show how existing usurpation
types detect the tribunal function before any topic-specific owner exists.

- Closed-canon veto: a received canon or interpretive closure functions as a veto over later divine speech. Route through FPD, V10/RT source-use discipline, and comparative-prophethood routes as appropriate before any prooftext content.
- Kashf-as-tribunal: an experience, shaykh, or tariqah claim is granted jurisdiction over revelation. Route authority order first; do not debate whether the experience occurred before classifying its authority.
- Vedic-reformist criterion: "reason/common sense" or Vedic reformist categories judge Qur'an, prophecy, attributes, resurrection, or law as an external tribunal. Route FPD + V2 + reason-disambiguation before verse-level response.
- Nondual category-set: Advaita-like nonduality or higher/lower truth hierarchy is installed above Islamic tawhid. Route M9 and metaphysical architecture before ordinary polytheism or devotional critique.

## Route Consequences

When usurpation is confirmed:

1. name the usurpation, not just the content claim
2. refuse to satisfy the tribunal before examining its jurisdiction
3. apply V2 to the framework itself
4. restore the correct authority ordering
5. engage the content question only after the framework has been moved from the chair to the table

---

## Prohibited Moves

- satisfying the usurping framework's requirements as the primary response
- accepting "God must be X" as a major premise and then defending revelation within it
- allegorizing revealed content to fit the framework before examining the framework's authority
- attacking the framework without restoring the tradition's positive authority ordering

---

## Restoration Targets

- relocate the framework from tribunal to contestable position
- restore the correct authority ordering
- only then engage the specific downstream philosophical objection with the matched content file
