# M5 — Deformation Triage

**Type:** Meta-tactic — run before any other tactic
**Deploy when:** Before selecting any tactic. This is the gateway meta-tactic.

Before selecting any tactic, identify which of the seven deformations is primary. The response appropriate for inherited doxastic rules (iʿtiqādāt mawrūtha) is categorically different from the response appropriate for vested interest (gharaḍ). Deploying argument against an entrenched will (hawā) deepens resistance. Deploying relational engagement against a genuine shubhah leaves the intellectual problem untouched.

**The triage sequence:**
1. Run the noetic reading checklist (`references/diagnostics/noetic-reading-checklist.md`) to surface the operative noetic structure
2. Identify which of the seven deformations (`references/diagnostics/seven-deformations.md`) is primary
3. Check discourse orientation (`references/diagnostics/discourse-orientation.md`) — if not truth-seeking, intellectual engagement may be the wrong instrument entirely
4. Select the tactic appropriate to the primary deformation

**The critical warning:** Presenting intellectual content to someone whose barrier is vested interest or hawā does not merely fail — it provides more material for resistance to organize around and actively hardens the barrier. The most common failure mode in sustained epistemological engagement is failing to triage — supplying arguments when the barrier is relational.

**The compound case:** Most real interlocutors exhibit more than one deformation. Triage the primary one; sequence the interventions from outside in (ʿāda before iʿtiqādāt mawrūtha; gharaḍ/hawā before any intellectual content).
