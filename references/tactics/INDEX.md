# Tactics — Index

Tactics are immediate conversational moves deployed responsively. Always run M5
(deformation triage, in this folder) before selecting a tactic.

## Organisation

Tactics are organized by register. Within each register, load the specific file for
the tactic needed. The meta-tactics (M1–M8) govern which register to enter and how
to approach it — load them before or alongside register tactics as needed.
