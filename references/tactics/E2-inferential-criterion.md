# E2 — Pressing the Inferential Criterion

**Register:** Evidentialist
**Deploy when:** The interlocutor holds that inferential argument is *necessary* for warranted theistic belief.

Interrogate the criterion itself: by what argument does one know that argument is required? This generates a regress.

**The key move:** The interlocutor's belief in other minds, the external world, the past, the reliability of memory — none of which rest on argument — shows that basic belief is already operating in their noetic structure. The criterion that disqualifies theistic belief equally disqualifies these. It proves too much.

The interlocutor must either:
- Relax the criterion (toward Soft Evidentialism — evidence may be required but is not the sole gateway)
- Acknowledge that basic belief is legitimate (toward Reformed Epistemology — theistic basic belief is back in contention)

**Connection:** Once the criterion is loosened, move to R1 (challenging the internalist criterion) for the Reformed register, or to E1 + E3 for the evidentialist register.
