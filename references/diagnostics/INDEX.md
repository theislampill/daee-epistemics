# Diagnostics — Index

This subfolder contains the analytical machinery for reading noetic structures, identifying
deformations and modes of concealment, and assessing discourse orientation. Load individual
files when the diagnostic protocol (SKILL.md §IV) requires their full treatment.

## Files

| File | Load When |
|------|-----------|
| `references/diagnostics/noetic-reading-checklist.md` | Running the nine-dimension internal analysis of any subject's noetic structure; personal ʿaqīdah input; any substantive engagement requiring structural diagnosis before response |
| `references/diagnostics/seven-deformations.md` | Full treatment of a deformation needed beyond the summary in SKILL.md §IV.D; selecting the precise intervention for a specific deformation; multiple deformations appear to be operative simultaneously |
| `references/diagnostics/four-modes-concealment.md` | Determining which mode of concealment is operative and what kind of engagement it requires; concealment mode governs whether intellectual engagement is appropriate at all |
| `references/diagnostics/discourse-orientation.md` | Discourse orientation is unclear or mixed; need precise characterization of what the engagement is ordered toward before selecting any tactic |
| `references/diagnostics/kalamic-interlocutor.md` | Interlocutor formed in kalāmic tradition; theistic belief held to require inferential demonstration; Māturīdī middle position |
| `references/diagnostics/fitrah-perspectives.md` | Interlocutor denies the fiṭrah is innate; holds blank-slate or dual-nature view; Muslim-internal dispute about the nature of the fiṭrah |

## Relationship to SKILL.md

SKILL.md §IV contains summary versions of the core diagnostic tools. The files here contain
the full treatment. The summary in SKILL.md is sufficient for clear-cut cases. Load the
relevant file when:
- The case is ambiguous or mixed
- The intervention requires precision beyond what the summary provides
- Multiple diagnoses appear to be simultaneously operative
