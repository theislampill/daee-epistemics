## Quick NS Identification

| Profile | Marker | Primary Deformation |
|---------|--------|---------------------|
| NS-1 Naturalist | "there is no evidence"; causal closure assumed | Iʿtiqādāt mawrūtha |
| NS-2 Agnostic Evidentialist | "belief must be proportioned to evidence" | Ẓann; possibly genuine shubhah |
| NS-3 Post-Religious | Deconversion narrative; orphaned moral commitments | Hawā; ẓann; possibly gharaḍ |
| NS-4 Secular Moral Realist | Moral facts are objective but God is rejected | Ẓann; iʿtiqādāt mawrūtha |
| NS-5 Convinced Atheist | God's non-existence held as basic; problem of evil load-bearing | Hawā; gharaḍ; ʿāda |
